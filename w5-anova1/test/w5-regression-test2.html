Во множественной линейной регрессии

  • нулевая гипотеза о равенстве всех коэффициентов
  • нулевая гипотеза проверяется при помощи t или F-критерия
  • для надежных выводов предикторы должны быть скоррелированы друг с другом
  • бета-коэффициенты можно сравнивать друг с другом

Случайные факторы (предикторы)

  • это такие, предикторы, возможные значения которых не были известны до начала исследования
  • при любом их значении можно предсказать значение зависимой переменной
  • это те, которые при получении дополнительной информации пришлось включить в анализ
  • для их обработки подходит регрессия по II модели

Методы подбора уравнения регрессии по II модели

  • OLS
  • MA
  • PCA
  • RMA

Как проходит линия регрессии по II модели?

  • между линиями регрессии x ~ y и y ~ x
  • параллельно линии регрессии x ~ y выше ее
