Отметьте правильные варианты ответов - Х а неправильные - 0

Коэффициенты простой линейной регрессии подбирают так, чтобы они минимизировали сумму квадратов

  • x (значений предиктора)
  • ε (остатков)
  • y (значений зависимой переменной)

Остатки вычисляются как

  • разница наблюдаемых и ожидаемых значений Y
  • предсказанное регрессией значение Y минус наблюдаемое значение Y
  • наблюдаемое значение X минус предсказанное регрессией значение X
  • наблюдаемое значение X минус предсказанное регрессией значение Y
